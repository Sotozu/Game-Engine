FieaGameEngine::SList< T >::SList ( const SList< T > &  right)

Copy constructor that initializes the SList<T> object with another SList<T> object. Calls IsEmpty() on the passed in SList<T> object to check if list is empty. If copying an empty SList<T> then the function calls Clear(). If copying an SList<T> that is NOT empty then then the function traverses the passed SList<T> objects contents. The calling SList<T> creates nodes for every Node in the passed in SList<T> object and deep copies them into respective Node member variable Data.
