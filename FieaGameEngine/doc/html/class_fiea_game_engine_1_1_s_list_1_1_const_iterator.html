Detailed Description

template<typename T>
class FieaGameEngine::SList< T >::ConstIterator

Used to traverse a list using an interface. ConstIterator points to an immutable item in the list it is assigned. Though we can still update the iterator (i.e., the iterator can be incremented or decremented but the item it points to can not be changed).
